# Hamilton Waterfall Explorer
Hamilton Waterfall Explorer is a web application that allows finding information about waterfalls located within the City of Hamilton, sorting the waterfalls by userâ€™s rating or the number of reviews and sharing own experiences by leaving reviews and ratings. This application is developed with HTML5, Bootstrap and uses a MySQL database to store information about waterfalls, user ratings and reviews.
